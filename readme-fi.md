# dit-doo
Given the last six letters of text, what will the next one be?

```
$ runhaskell dit-doo-cks-012.hs 

MATKUSTUS MAAN KESKIPISTEESEEN

Kirjoitettuna, Bacon, Lullus ja Milne Edvards keskuudessa. Ja silloin, kun 
äkille!

Setäni viittä tarkasteli hän.

Ja hyvää kuin nuuskaa, mutta en voi kylliksi useita, nimittäin lausua, tuolla 
niin ihmetelin vaihteli ihastuetelämässä, niin sen lisäkamariinkin, mutistä, 
miten tarkka se ei siis itseksi, nusin hänen huoneestä välittää?

Katseen rauhaansa olisin järjesteli joskus erehtyy.

Kaupunginosan halainen, ulkona leikkiä lyöneet islannissani. Hän oli maisesteet 
kirjakulu, ja sen näin vastahakoisen arvelematta määrätä keksin?

-- Etkä sinä pöllö, että professori Otto Lidenbrock ei ole vierlantilaailmassa 
avasikanselko tämän kirjoitettu oikeata luki:

[Kuva: Riimukirjaimilla?

-- Mutta kuitenkin oikeassa järjestykseen saksalaisena ihmineralogisia tämän 
kirjoitettu tuon usein liian maltittomimpia Hampuristetyssä niin olin moni 
oppinut egoisti, tieteet, jotka harrastaan saakka, joka värikun ne mahtanevat 
monta tärkeimmista, retinasfaltista, antrasiitä selvän.

-- Olkoon minkin valloinä.
```
