Given the last six letters of text, what will the next one be?

Collecting chunks

Lets first collect all the chunks of length [7,6..1] in a given text.

str = "sinus aestuum"

chunks n xs 
  | n <= length xs = fst (splitAt n xs) : chunks n (tail xs)
  | otherwise      = []

chks = [chunks n str | n <- [7,6..1]]

main = do
  mapM_ print chks
